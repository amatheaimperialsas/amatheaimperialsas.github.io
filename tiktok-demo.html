TikTok API Demo — SANDBOX

1️⃣ Login Kit — Autorización OAuth (user.info.basic)

Conecta la cuenta TikTok autorizada del sandbox. Serás redirigido a TikTok para aprobar los permisos y volverás a esta página con el código de autorización.

2️⃣ Perfil de la cuenta autorizada (user.info.basic)

Con el token de acceso obtenido, consultamos el perfil básico de la cuenta autorizada mediante la API oficial.

3️⃣ Content Posting — Direct Post (video.publish)

Publica directamente un video promocional de Amathea Imperial en la cuenta autorizada (privacidad SELF_ONLY en sandbox).

4️⃣ Content Posting — Upload como borrador (video.upload)

Sube el mismo video a la bandeja de borradores del creador para editar y publicar desde la app de TikTok.
